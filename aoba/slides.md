## 【第1回】スマホの基本操作＆設定

### 1. スマホの基本的な使い方（電源）
- iPhone：電源ボタン＋音量ボタン長押しで電源オフ、電源ボタン長押しでオン  
- Android：電源ボタン長押しでオン／オフ

![Alt text](ダウンロード/Gemini_Generated_Image_sqganysqganysqga.png "a title")

### 2. 指の使い方
- タップ：軽く触って離す  
- 長押し：押し続ける  
- スワイプ：指を滑らせる  
- ピンチ：2本指で拡大・縮小

### 3. ホーム画面
- iPhone：左へスワイプでアプリ一覧  
- Android：下から上へスワイプでアプリ一覧  
- アプリ追加：アイコンを長押しして上へ移動

### 4. 戻る・ホーム・起動中アプリ
- iPhone：画面端からスワイプ、下から上でアプリ一覧  
- Android：戻る／ホーム／履歴ボタン

### 5. アプリ管理
- インストール：iPhone＝App Store、Android＝Playストア  
- アンインストール：アイコン長押し → 削除  
- フォルダ作成：アイコンを重ねる

### 6. スマホ設定
- 通知、音量、明るさ、文字サイズ、バッテリー、Wi-Fi、パスワード管理など

## 【第2回】電話と連絡先をマスターしよう

### 1. 電話のかけ方
- 電話番号を入力して発信  
- 着信履歴から発信  
- 連絡先（電話帳）から発信

### 2. 電話の受け方
- 着信画面で応答  
- 終了は「通話終了」

### 3. 不在着信のかけ直し
- 電話アプリ → 着信履歴 → 不在着信をタップ → 発信

### 4. 連絡先（電話帳）の基本
- 登録できる情報：番号、メール、住所、誕生日など  
- 表示方法：電話アプリの「連絡先」タブ or 連絡先アプリ

### 5. 連絡先の登録・編集
- 新規登録：連絡先 → 追加 → 入力 → 保存  
- 着信履歴から登録：履歴 → 番号 →「連絡先に追加」  
- 編集：連絡先 → 編集 → 保存  
- 削除：連絡先 → メニュー → 削除

## 【第3回】文字入力マスター講座

### 1. キーボードの基本
- 文字入力欄をタップするとキーボードが表示  
- テンキー／QWERTY／手書き入力が選べる  
- 切替：iPhone＝地球儀、Android＝☺記

### 2. トグル入力（ケータイ入力）
- 同じキーを連続タップして「あ→い→う→え→お」

### 3. フリック入力
- 指を上下左右に払って入力  
- 例：「い」＝「あ」のキーを左へ払う

### 4. 音声入力
- キーボードのマイクをタップ  
- 「まる」「てん」「かいぎょう」で句読点・改行  
- 騒音があると認識しにくい

### 5. 文字の修正
- 削除：×  
- カーソル移動：長押しして動かす  
- コピー＆ペースト：長押し → 範囲選択 → コピー／ペースト

### 6. 効率化テクニック
- 予測変換  
- 定型文（ユーザ辞書）  
- 両手入力・横画面入力

### 6. QRコード読み取り
- 基本はカメラでOK  
- らくらくスマホは専用メニューあり

### 7. Googleレンズ

- 文字読み取り  
- 翻訳

## 【第9回】インターネット検索をしてみよう

### 1. ブラウザとは
- Chrome（Android）  
- Safari（iPhone）  
- ニュース・調べ物・Web閲覧に使用

### 2. 広告ブロック
- Chrome：設定 → サイトの設定 → ポップアップをオフ  
- Safari：設定 → Safari → ポップアップブロック

### 3. 検索の基本
- キーワード入力  
- 音声検索  
- Googleレンズで画像検索  
- before: / after: で期間指定検索

### 4. 安全なネット利用
- 個人情報を入力しない  
- 詐欺サイトに注意  
- メールのリンクを不用意に押さない
